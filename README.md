# # Portfolio of Vishal Vashishth

This repository contains the source code for the personal portfolio website of Vishal Vashishth. The project is built with React, Vite, Tailwind CSS, and Framer Motion.

## About

- Full Stack Web Developer portfolio showcasing skills, projects, and contact details.
- Includes project links, social profiles, and resume download support.

## Tech stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Icons

Open the browser at `https://vishalvashishth.netlify.app/` to preview the portfolio.
